Convert any HTML to a PDF using any valid HTML, CSS, Javascript, SASS, etc. As long as the base64 encoded JSON payload is below 10MB in size, we will accept it and convert it.

## Endpoints

### POST /html/pdf

Supports all parameters being passed into request as JSON. The following parameter types are supported:

- [Html Parameters](#html-parameters)
- [Base Parameters](#base-parameters)
- [Pdf Parameters](#pdf-parameters)

```html title="POST /html/pdf"
https://api.cloudlayer.io/v1/html/pdf
```

```json title="JSON Payload"
{ "html": "PGh0bWw+PGJvZHk+PGgxPkhlbGxvIHdvcmxkITwvaDE+PC9ib2R5PjwvaHRtbD4=" }
```

### Examples

```json title="Viewport Example"
{
  "html": "PGh0bWw+PGJvZHk+PGgxPkhlbGxvIHdvcmxkITwvaDE+PC9ib2R5PjwvaHRtbD4=",
  "viewPort": {
    "width": 800,
    "height": 400
  }
}
```

```shell title="cURL Example"
curl --request POST \
  --url https://api.cloudlayer.io/v1/html/pdf \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <YOUR-API-KEY>' \
  --data '{
	"html": "PGh0bWw+PGJvZHk+PGgxPkhlbGxvIHdvcmxkITwvaDE+PC9ib2R5PjwvaHRtbD4="
}' \
--output "google.pdf"
```

```shell title="Wget Example"
wget --method POST \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <YOUR-API-KEY>' \
  --body-data '{"html": "PGh0bWw+PGJvZHk+PGgxPkhlbGxvIHdvcmxkITwvaDE+PC9ib2R5PjwvaHRtbD4="}' \
  - https://api.cloudlayer.io/v1/html/pdf \
  -O "google.pdf"
```

:::tip
Make sure your `Content-Type` gets set to `application/json`, and that you have placed your `X-API-Key` header with your API Key from your account.
:::
