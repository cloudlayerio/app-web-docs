## Base Parameters

Parameters that exist on **all** services.

### `waitUntil`

When to consider the navigation has been completed and begin the conversion.

`string`
_`Default: "networkidle2"`_

#### Accepted Values

- `load` - consider navigation to be finished when the `load` event is fired.
- `domcontentloaded` - consider navigation to be finsihed when the `DOMContentLoaded` event is fired.
- `networkidle0` - consider navigation to be finished when there are no more than 0 network connections for at least `500` ms.
- `networkidle2` - consider navigation to be finished when there are no more than 2 network connections for at least `500' ms.

```json
{ "waitUntil": "load" }
```

### `timeout`

Amount of time that Chrome is allowed to run. If exceeded, your job will terminate.

`number`
_`Default: 30000ms`_

```json
{ "timeout": 30000 }
```

### `delay`

The amount of time in milliseconds to wait for the page to complete rendering before conversion.

`number`
_`Default: 0ms`_

```json
{ "delay": 1000 }
```

### `autoScroll`

Will auto-scroll the page down to the very end. Helpful in forcing lazy-loaded content to load.

`boolean`
_`Default: false`_

```json
{ "autoScroll": true }
```

### `filename`

If used with the `inline: false`, will set the `Content-Disposition` filename so that the downloaded file will assign to this value in the user's browser. For `inline: true`, it has no effect.

`string`
_`Default: ""`_

```json
{ "filename": "filename.ext" }
```

### `inline`

If set to true, the `Content-Disposition` gets set to `inline`. If set to false, it gets set to `attachment`. See [filename](#filename) property if you want to set the filename value for the attachment.

`boolean`
_`Default: false`_

```json
{ "inline": true }
```

### `projectId`
The project's id you would like the job and any assets the job generates to be a member.

`string`
_`Default: ""`_

```json
{ "projectId": "N77VCoAVTHHwgmUfCYlD" }
```

### `viewPort`

Set the Viewport of the page before navigation.

```json
{
  "viewPort": {
    "width": 1024,
    "height": 768,
    "deviceScaleFactor": 2
  }
}
```

#### `width` {#viewport-width}

Page width in pixels only.
**Required**

`number`
_`Default: <null>`_

```json
{
  "viewPort": {
    "width": 1024
  }
}
```

#### `height` {#viewport-height}

Page height in pixels only.
**Required**

`number`
_`Default: <null>`_

```json
{
  "viewPort": {
    "height": 768
  }
}
```

#### `deviceScaleFactor`

Specify device scale factor (DPR). [Height](#viewport-height) and [Width](#viewport-width) are always required.

`number`
_`Default: 1`_

```json
{
  "viewPort": {
    "width": 640,
    "height": 480,
    "deviceScaleFactor": 1
  }
}
```

#### `isMobile`

Whether the `meta viewport` tag is taken into account. [Height](#viewport-height) and [Width](#viewport-width) are always required.

`boolean`
_`Default: false`_

```json
{
  "viewPort": {
    "width": 640,
    "height": 480,
    "isMobile": true
  }
}
```

#### `hasTouch`

Specifies if viewport supports touch events. [Height](#viewport-height) and [Width](#viewport-width) are always required.

`boolean`
_`Default: false`_

```json
{
  "viewPort": {
    "width": 640,
    "height": 480,
    "hasTouch": true
  }
}
```

#### `isLandscape`

Specifies if viewport is in landscape mode. [Height](#viewport-height) and [Width](#viewport-width) are always required.

`boolean`
_`Default: false`_

```json
{
  "viewPort": {
    "width": 640,
    "height": 480,
    "isLandscape": true
  }
}
```

### `timeZone`

Changes the timezone of the page. See [ICU's metaZones.txt](https://source.chromium.org/chromium/chromium/deps/icu.git/+/faee8bc70570192d82d2978a71e2a615788597d1:source/data/misc/metaZones.txt) for a list of supported timezone IDs.

`string`
_`Default: <null>`_

```json
{ "timeZone": "Europe/Rome" }
```

### `waitForSelector`

Allows for more complex wait behavior to control when the Chrome rendering engine decides when a page has finished before the conversion.

#### `selector`

The selector of an element to wait for before performing the conversion.

`string`
_`Default: <null>`_

```json
{
  "waitForSelector": {
    "selector": "img"
  }
}
```

#### `options`

The selector of an element to wait for before performing the conversion.

#### `visible`

Wait for the [selector](#selector) element to be present in the DOM and to be visible, i.e. to not have `display:none` or `visibility:hidden` CSS properties. Not to be combined with the [hidden](#hidden) property.

`boolean`
_`Default: false`_

```json
{
  "waitForSelector": {
    "selector": "img",
    "options": {
      "visible": true
    }
  }
}
```

#### `hidden`

Wait for the [selector](#selector) element to be present in the DOM and to be hidden, i.e. to have `display: none` or `visibility: hidden` CSS properties. Not to be combined with the [visible](#visible) property.

`boolean`
_`Default: false`_

```json
{
  "waitForSelector": {
    "selector": "img",
    "options": {
      "hidden": true
    }
  }
}
```

#### `timeout`

Maximum amount of time to wait in milliseconds for the selector to meet the [visible](#visible) or [hidden](#hidden) criteria.

`number`
_`Default: 30000`_

```json title="visible"
{
  "waitForSelector": {
    "selector": "img",
    "options": {
      "visible": true,
      "timeout": 30000
    }
  }
}
```

```json title="hidden"
{
  "waitForSelector": {
    "selector": "img",
    "options": {
      "hidden": true,
      "timeout": 30000
    }
  }
}
```

### `preferCSSPageSize`

Give any CSS `@page` size declared in the page priority over what is declared in [width](#width) and [height](#height) or [format](#format) options.

`boolean`
_`Default: false`_

```json
{ "preferCSSPageSize": true }
```

### `scale`

Scale of the webpage rendering. Must be between 0.1 and 2.

`number`
_`Default: 0`_

```json
{ "scale": 0.5 }
```

### `height`

Paper height. Must be supported type of [units](#units) or number.

`string|number`
_`Default: <null>`_

```json
{ "height": 100 }
```

```json
{ "height": "1.25in" }
```

### `width`

Paper width. Must be supported type of [units](#units) or number.

`string|number`
_`Default: <null>`_

```json
{ "width": 100 }
```

```json
{ "width": "1.25in" }
```

### `landscape`

Paper orientation, false sets it to portrait and true to landscape.

`boolean`
_`Default: false_

```json
{ "landscape": true }
```
