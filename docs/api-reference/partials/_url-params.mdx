## Url Parameters

Parameters that exist on **URL** services.

### `authentication`

Provide credentials for [HTTP authentication](https://developer.mozilla.org/en-US/docs/Web/HTTP/Authentication) (Basic Auth).

```json
{
  "authentication": {
    "username": "bob",
    "password": "abc123"
  }
}
```

### `cookies`

Sets up one to many cookies for the page prior to navigation.

```json
{
  "cookies": [
    {
      "name": "cookie_notice_accept",
      "value": "1",
      "domain": "www.example.com",
      "path": "/",
      "expires": 1668752280
    },
    {
      "name": "_session_id",
      "value": "ad9a8u90f0df7d87fdas9fa892342",
      "domain": "www.example.com",
      "path": "/",
      "expiration": 1668752280,
      "httpOnly": true
    }
  ]
}
```

#### `name` {#cookie-name}

The name of the cookie to set. Name and value are both required.
**Required**

`string`
_`Default: ""`_

```json
{
  "cookies": [
    {
      "name": "myCookie",
      "value": "..."
    }
  ]
}
```

#### `value` {#cookie-value}

The name of the cookie to set. Name and value are both required.
**Required**

`string`
_`Default: ""`_

```json
{
  "cookies": [
    {
      "name": "...",
      "value": "SomeCookieValue"
    }
  ]
}
```

#### `url` {#cookie-name}

The url to use for the cookie. This is usually left empty.

`string`
_`Default: ""`_

```json
{
  "cookies": [
    {
      "name": "myCookie",
      "value": "some_data",
      "url": "https://www.example.com"
    }
  ]
}
```

#### `domain` {#cookie-domain}

Defines the host to which the cookie will be sent. If omitted, this attribute defaults to the host of the current document URL, not including subdomains.

`string`
_`Default: ""`_

```json
{
  "cookies": [
    {
      "name": "myCookie",
      "value": "some_data",
      "domain": "www.example.com"
    }
  ]
}
```

#### `path` {#cookie-path}

Indicates the path that must exist in the requested URL for the browser to send the `Cookie` header.

`string`
_`Default: ""`_

```json
{
  "cookies": [
    {
      "name": "myCookie",
      "value": "some_data",
      "path": "/"
    }
  ]
}
```

#### `expires` {#cookie-expires}

Indicates the maximum lifetime of the cookie as an Unix time integer.

`number`
_`Default: <null>`_

```json
{
  "cookies": [
    {
      "name": "myCookie",
      "value": "some_data",
      "expires": 1668752280
    }
  ]
}
```

#### `httpOnly` {#cookie-httponly}

Forbids JavaScript from accessing the cookie.

`boolean`
_`Default: <null>`_

```json
{
  "cookies": [
    {
      "name": "myCookie",
      "value": "some_data",
      "httpOnly": true
    }
  ]
}
```

#### `secure` {#cookie-secure}

Indicates that the cookie is sent to the server only when a request is made with the https: scheme.

`boolean`
_`Default: <null>`_

```json
{
  "cookies": [
    {
      "name": "myCookie",
      "value": "some_data",
      "secure": true
    }
  ]
}
```

#### `sameSite` {#cookie-samesite}

Controls whether or not a cookie is sent with cross-origin requests.

`string<"Strict|Lax">`
_`Default: <null>`_

Values

- `Strict`
- `Lax`

```json
{
  "cookies": [
    {
      "name": "myCookie",
      "value": "some_data",
      "sameSite": "Strict"
    }
  ]
}
```

### `url`

Source URL to use for generating the content.

**Required**

`string`
_`Default: ""`_

```json
{ "url": "http://google.com" }
```

#### `username`

The username for authenticating. Username and Password are both required.

**Required**

`string`
_`Default: ""`_

```json
{
  "authentication": {
    "username": "bob",
    "password": "..."
  }
}
```

#### `password`

The password for authenticating. Username and Password are both required.

**Required**

`string`
_`Default: ""`_

```json
{
  "authentication": {
    "username": "...",
    "password": "abc123"
  }
}
```
